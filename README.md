# Around The U.S.

This is the third project of the Software Engineering program at TripleTen. It was created using HTML and CSS, based on the design provided on Figma.

## Project features

- Semantic HTML5
- Flexbox
- Grid
- Flat BEM file structure
- Responsive design
- JavaScript
