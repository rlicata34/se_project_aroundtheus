# Around The U.S.

This is a project from the Software Engineering program at TripleTen. It was created using HTML, CSS and Javascript based on the design provided on Figma.

## Project features

- Semantic HTML5
- Flexbox
- Grid
- Flat BEM file structure
- Responsive design
- JavaScript
